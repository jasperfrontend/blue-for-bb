# Blue for BB

A cloud sync solution for Beaver Builder templates, rows, columns, and modules. Seamlessly backup, restore, and share your Beaver Builder components across multiple WordPress installations.

## Features

- **Cloud Backup** - Automatically sync your Beaver Builder templates to the cloud
- **Easy Restore** - Quickly restore templates, rows, columns, and modules
- **Cross-Site Sharing** - Share components across your WordPress installations
